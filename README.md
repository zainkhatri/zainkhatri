## Featured Projects

### NASA Autonomous Rover  
This project involves building an autonomous rover for navigation using GPS and magnetometer data. The rover was developed using Arduino, programmed in C, with extensive work on wiring, soldering, and 3D printing custom parts.  
[View Project](https://github.com/zainkhatri/nasa-caspacegrant-autonomous-rover)

### UC Berkeley Exoskeleton Brainwave Algorithm  
This algorithm uses EEG data and machine learning to control an exoskeleton, helping individuals with stroke or Parkinson's improve mobility and stability. By detecting left or right leaning in under 30 milliseconds with 90% accuracy, it provides real-time feedback to correct posture.  
[View Project](https://github.com/zainkhatri/exoskeleton-brainwave-algorithm)

### MunchMate - AI Recipe Generator  
An AI-powered recipe generator that transforms available ingredients into healthy, delicious meals. Built with React and OpenAI's GPT-3.5, it provides personalized recipes with nutritional information and workout plans.  
[View Project](https://github.com/zainkhatri/munchmate)

### Eagle Eye Game  
Interactive game powered by machine learning where players must quickly find specific targets on a grid. The application demonstrates practical applications of vision processing algorithms in an engaging format.  
[View Project](https://github.com/zainkhatri/eagle-eye-game)
